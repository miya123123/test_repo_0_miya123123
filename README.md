# Color Chain Battle - 2画面対戦パズルゲーム

## 🎮 ゲーム概要

Color Chain Battleは2つの画面で同時にプレイする斬新な対戦型パズルゲームです。プレイヤーは同じ色のブロックを繋げてチェーンを作り、スコアを競います。

## 🎯 ゲームの特徴

- **2画面同時プレイ**: 左右の画面で2人が同時に対戦
- **チェーンシステム**: 同じ色のブロックを3つ以上つなげてチェーンを作成
- **特殊攻撃**: 相手の画面に妨害ブロックを送る戦略的要素
- **リアルタイム対戦**: 3分間の制限時間内で高得点を競う
- **美しいビジュアル**: 現代的でカラフルなUI/UXデザイン

## 🎲 遊び方

1. **ゲーム開始**: 「Game Start」ボタンをクリック
2. **チェーン作成**: 同じ色のブロックを3つ以上つなげてクリック
3. **得点獲得**: チェーンが長いほど高得点
4. **特殊攻撃**: 「Special Attack」ボタンで相手を妨害
5. **勝利条件**: 3分間で相手より高いスコアを獲得

## 🎯 得点システム

- **基本得点**: 3つのブロック = 10点
- **チェーンボーナス**: 4つ目以降 = +5点/ブロック
- **例**: 5つのチェーン = 10 + 5 + 5 = 20点

## 💣 特殊攻撃

- **効果**: 相手の画面にグレーの妨害ブロックを8個配置
- **クールダウン**: 5秒間使用不可
- **戦略**: タイミングを見計らって相手を妨害
